# เทคนิคและการใช้งานของ Machine Learning

## วัตถุประสงค์การเรียนรู้
หลังจากศึกษาเนื้อหานี้ คุณจะสามารถ
- อธิบายการเรียนรู้ของเครื่อง (Machine Learning)
- ระบุเทคนิคการเรียนรู้ของเครื่อง
- อธิบายการใช้งานของการเรียนรู้ของเครื่อง

---

## 1. ปัญญาประดิษฐ์และ Machine Learning

### ปัญญาประดิษฐ์ (Artificial Intelligence - AI)
AI ทำให้คอมพิวเตอร์ดูฉลาดโดยจำลองความสามารถทางปัญญาของมนุษย์ 

AI เป็นสาขาทั่วไปที่มีขอบเขตกว้าง ประกอบด้วย
- วิสัยทัศน์คอมพิวเตอร์ (Computer Vision)
- การประมวลผลภาษาธรรมชาติ (Natural Language Processing)
- AI แบบสร้างสรรค์ (Generative AI)
- การเรียนรู้ของเครื่อง (Machine Learning)
- การเรียนรู้เชิงลึก (Deep Learning)

### Machine Learning (ML)
Machine Learning เป็นชุดย่อยของ AI ที่มีลักษณะดังนี้
- ใช้อัลกอริทึมในการเรียนรู้
- ต้องใช้วิศวกรรมคุณลักษณะ (Feature Engineering) โดยผู้ปฏิบัติงาน
- สอนคอมพิวเตอร์ให้เรียนรู้จากข้อมูล ระบุรูปแบบ และตัดสินใจ โดยไม่ได้รับคำแนะนำที่ชัดเจนจากมนุษย์

### Deep Learning
การเรียนรู้เชิงลึก แตกต่างจาก ML แบบดั้งเดิมโดย
- ใช้เครือข่ายประสาทเทียมหลายชั้น (Multi-layer Neural Networks)
ดึงคุณสมบัติจากข้อมูลขนาดใหญ่ที่ซับซ้อนสูงและไม่มีโครงสร้างโดยอัตโนมั

2. วิธีการเรียนรู้ของ Machine Learning

2.1 การเรียนรู้แบบมีการกำกับดูแล (Supervised Learning)
- ฝึกอบรม ใช้ข้อมูลที่มีป้ายกำกับ (Labeled Data)
- เป้าหมาย  เรียนรู้วิธีอนุมานเกี่ยวกับข้อมูลใหม่
- ผลลัพธ์ สามารถทำนายป้ายกำกับที่ไม่รู้จัก

### 2.2 การเรียนรู้แบบไม่มีการกำกับดูแล (Unsupervised Learning)
- ฝึกอบรม ทำงานโดยไม่มีป้ายกำกับ
- เป้าหมาย ค้นหารูปแบบในข้อมูล

### 2.3 การเรียนรู้แบบกึ่งกำกับดูแล (Semi-supervised Learning)
- ฝึกอบรม ใช้ข้อมูลชุดย่อยที่ค่อนข้างเล็กซึ่งมีการติดฉลากแล้ว
- กระบวนการ ฝึกซ้ำอีกครั้งโดยเพิ่มป้ายกำกับใหม่ที่สร้างขึ้นด้วยความมั่นใจสูงพอสมควร

### 2.4 การเรียนรู้เสริมกำลัง (Reinforcement Learning)
- กระบวนการ จำลองตัวแทนอัจฉริยะเทียมที่มีปฏิสัมพันธ์กับสภาพแวดล้อม
- การเรียนรู้ เรียนรู้วิธีตัดสินใจตามข้อเสนอแนะจากสภาพแวดล้อม

## 3. เทคนิค Machine Learning

### การเลือกเทคนิค
ขึ้นอยู่กับปัจจัยหลายประการ
- ปัญหาที่คุณพยายามแก้ไข
- ประเภทของข้อมูลที่คุณมี
- ทรัพยากรที่มีอยู่
- ผลลัพธ์ที่ต้องการ

### 3.1 การจำแนกประเภท (Classification)
วัตถุประสงค์ ทำนายประเภทหรือหมวดหมู่ของกรณี

ตัวอย่าง
- เซลล์ไม่เป็นอันตรายหรือร้าย
- ลูกค้าจะหมดหรือไม่

### 3.2 การถดถอย (Regression)
วัตถุประสงค์ ทำนายค่าอย่างต่อเนื่อง

ตัวอย่าง
- ราคาของบ้านตามลักษณะต่างๆ
- การปล่อยก๊าซคาร์บอนไดออกไซด์จากเครื่องยนต์รถ

### 3.3 การจัดกลุ่ม (Clustering)
วัตถุประสงค์ จัดกลุ่มกรณีที่คล้ายกัน

ตัวอย่าง
- ค้นหาผู้ป่วยที่คล้ายกัน
- การแบ่งกลุ่มลูกค้าในสาขาการธนาคาร

### 3.4 การเชื่อมโยง (Association)
วัตถุประสงค์ ค้นหาสิ่งของหรือเหตุการณ์ที่มักเกิดขึ้นร่วมกัน

ตัวอย่าง
- สินค้าของชำที่มักจะซื้อด้วยกัน
- กลุ่มตลาดเฉพาะ

### 3.5 การตรวจจับความผิดปกติ (Anomaly Detection)
วัตถุประสงค์ ค้นพบกรณีที่ผิดปกติและผิดปกติ

ตัวอย่าง
- การตรวจจับการฉ้อโกงด้วยบัตรเครดิต

### 3.6 การขุดลำดับ (Sequence Mining)
วัตถุประสงค์ ทำนายเหตุการณ์ถัดไป

ตัวอย่าง
- การวิเคราะห์คลิกสตรีมในเว็บไซต์

### 3.7 การลดมิติข้อมูล (Dimensionality Reduction)
วัตถุประสงค์ ลดขนาดข้อมูล โดยเฉพาะจำนวนคุณสมบัติที่จำเป็น

### 3.8 ระบบแนะนำ (Recommendation System)
วัตถุประสงค์ เชื่อมโยงความชอบของผู้คนกับผู้อื่นที่มีรสนิยมคล้ายกัน

ตัวอย่าง
- แนะนำหนังสือ
- แนะนำภาพยนตร์

## 4. การเปรียบเทียบเทคนิคสำหรับ Supervised Learning

### การจำแนกประเภท (Classification)
- จำแนกข้อมูลอินพุตเป็นหนึ่งในหมวดหมู่หรือคลาสที่กำหนดไว้ล่วงหน้า
- ทำการคาดการณ์เกี่ยวกับสมาชิกคลาสของการสังเกตใหม่

### การถดถอย (Regression)
- แตกต่างจากการจำแนกประเภท
- ไม่ได้ทำนายเกี่ยวกับการเป็นสมาชิกคลาส
- ทำนายค่าตัวเลขต่อเนื่องจากข้อมูลอินพุต

### การคลัสเตอร์ (Clustering) - Unsupervised
- เทคนิคการเรียนรู้ของเครื่องที่ไม่มีการกำกับดูแล
- ใช้เพื่อจัดกลุ่มจุดข้อมูลหรือวัตถุที่คล้ายคลึงกัน
- จุดข้อมูลที่ไม่มีหมวดหมู่ถือเป็นเสียงรบกวน (Noise)

## 5. กรณีศึกษา การตรวจจับเซลล์มะเร็ง

### ปัญหา
มีตัวอย่างเซลล์มนุษย์ที่สกัดจากผู้ป่วยซึ่งมีลักษณะเฉพาะ เช่น
- ความหนาของก้อน (Clump Thickness) 6
- ความสม่ำเสมอของขนาดเซลล์ (Uniformity) 1
- การยึดเกาะขอบ (Marginal Adhesion) 1

คำถาม เซลล์นี้เป็นเซลล์ที่อ่อนโยน (Benign) หรือมะเร็ง (Malignant)?

### ความสำคัญ
- เซลล์มะเร็งบุกรุกเนื้อเยื่อโดยรอบและแพร่กระจายไปทั่วร่างกาย
- การระบุประเภทเซลล์ในช่วงต้นอาจเป็นกุญแจสำคัญในการอยู่รอดของผู้ป่วย

### แนวทางแก้ปัญหาด้วย Machine Learning

#### 1. เตรียมข้อมูล
- รวบรวมชุดข้อมูลที่มีลักษณะของเซลล์มนุษย์หลายพันตัวอย่าง
- ข้อมูลจากผู้ป่วยที่เชื่อว่ามีความเสี่ยงต่อการเป็นมะเร็ง

#### 2. วิเคราะห์ข้อมูล
- ลักษณะหลายอย่างแตกต่างกันอย่างมีนัยสำคัญระหว่างตัวอย่างที่ไม่ดีและมะเร็ง

#### 3. สร้างโมเดล
- ทำความสะอาดข้อมูล
- เลือกอัลกอริทึมที่เหมาะสมสำหรับการสร้างแบบจำลองการทำนาย
- ฝึกโมเดลให้เข้าใจรูปแบบของเซลล์ที่ไม่ดีหรือเซลล์มะเร็งภายในข้อมูล

#### 4. ใช้งานโมเดล
- เมื่อโมเดลได้รับการฝึกฝนโดยผ่านข้อมูลซ้ำแล้ว
- สามารถใช้ทำนายเซลล์ใหม่หรือไม่รู้จักด้วยความแม่นยำที่ค่อนข้างสูง

> นี่คือพลังของการเรียนรู้ของเครื่อง

## 6. การใช้งาน Machine Learning ในชีวิตจริง

### 6.1 ระบบแนะนำ (Recommendation Systems)
Amazon และ Netflix
- แนะนำเนื้อหาและผลิตภัณฑ์สำหรับผู้ใช้
- คล้ายกับวิธีที่เพื่อนแนะนำรายการโทรทัศน์ตามความรู้เกี่ยวกับประเภทที่คุณชอบ

### 6.2 การอนุมัติสินเชื่อ (Credit Approval)
ธนาคาร
- ใช้ ML เพื่อทำนายความน่าจะเป็นในการผิดนัดชำระของผู้สมัครแต่ละคน
- ตัดสินใจอนุมัติหรือปฏิเสธใบสมัครสินเชื่อตามความน่าจะเป็นนั้น

### 6.3 การทำนายการหมดอายุของลูกค้า (Churn Prediction)
บริษัทโทรคมนาคม
- ใช้ข้อมูลประชากรเพื่อแบ่งกลุ่มลูกค้า
- คาดการณ์ว่าลูกค้าจะยกเลิกการสมัครสมาชิกภายในเดือนข้างหน้าหรือไม่

### 6.4 วิสัยทัศน์คอมพิวเตอร์ (Computer Vision)

#### ปัญหา แยกแมวออกจากสุนัข
วิธีการแบบเดิม
- ต้องตีความภาพว่าเป็นชุดของคุณสมบัติ
  - มีดวงตาหรือไม่? ขนาดเท่าไร?
  - มีหูหรือไม่?
  - มีหางหรือไม่?
  - กี่ขา?
  - มีปีกหรือไม่?
- ต้องสร้างกฎอย่างพิถีพิถัน
- ข้อจำกัด
  - ต้องการกฎจำนวนมาก
  - ขึ้นอยู่กับชุดข้อมูลปัจจุบันมากเกินไป
  - ไม่สามารถระบุกรณีที่ไม่เห็นได้

วิธีการด้วย Machine Learning
- สร้างแบบจำลองที่เรียนรู้ชุดคุณสมบัติของสัตว์ที่โดดเด่น
- เรียนรู้จากตัวอย่างที่รู้จัก
- สามารถอนุมานประเภทสัตว์โดยอัตโนมัติ

### 6.5 แอปพลิเคชันอื่นๆ
- ผู้ช่วยเสมือน แชทบอท
- การจดจำใบหน้า ปลดล็อคโทรศัพท์
- เกมคอมพิวเตอร์ เล่นหมากรุก

## 7. บทบาทของมนุษย์ใน Machine Learning

แม้ว่าแอปพลิเคชัน ML ในโลกสมัยใหม่จะเพิ่มขึ้น แต่มนุษย์ยังคงมีบทบาทสำคัญ

ตัวอย่าง
- หากอัลกอริทึม ML ปฏิเสธใบสมัครสินเชื่อ
- ธนาคารต้องการทราบว่าทำไม
- ต้องมีการดำเนินการติดตาม
